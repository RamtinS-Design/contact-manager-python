# Contact Manager (Python CLI)
A simple Contact Manager built with Python.
This program allows users to store, view, search, and delete contacts using a dictionary-based system.

## Features 
- Add new contacts
- View all saved contacts
- Search contacts by name
- Delete contacts
- Simple command-line interface

## How it Works
- Press Num 1 to 5
- 1 For Adding
- 2 For Viewing
- 3 For Searching
- 4 For Deleting
- 5 For Exiting
